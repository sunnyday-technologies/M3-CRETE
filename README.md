# M3-CRETE — Open-Source Concrete 3D Printer

An open-source concrete 3D printing system developed by [Sunnyday Technologies](https://sunnydaytech.com).

## About

M3-CRETE is a community-driven, open-source concrete 3D printer designed to make additive construction accessible. This repository contains hardware designs, firmware configurations, tools, and documentation for building and operating the system.

## Repository Structure

```
m3-crete/
├── configurator/    — Browser-based printer configurator
├── firmware/        — Printer firmware and configurations
├── hardware/        — CAD files, BOM, mechanical drawings
├── docs/            — Build guides, calibration, specifications
└── images/          — Photos and diagrams
```
